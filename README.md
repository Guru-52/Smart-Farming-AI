# 🌾 Smart Farming AI

## 📌 Project Overview
Smart Farming AI is an AI-powered agricultural assistant designed to support farmers in making smarter and data-driven decisions. It combines Machine Learning and Retrieval-Augmented Generation (RAG) to recommend suitable crops based on environmental conditions and provide intelligent answers to farming-related queries.

The system leverages agricultural datasets and a vector database (Endee) to deliver accurate predictions and context-aware responses, improving productivity and reducing farming risks.

---

## ✨ Key Features

### 🌱 Crop Recommendation Engine
- Predicts the most suitable crop based on environmental inputs:
  - Temperature
  - Humidity
  - Rainfall
  - Soil type
  - Season
- Uses trained Machine Learning models for prediction accuracy

### 🤖 AI Farming Chat Assistant
- Answers farmer queries in real time
- Uses RAG (Retrieval Augmented Generation)
- Vector database (Endee) stores agricultural knowledge
- Provides guidance on:
  - Fertilizers
  - Pest control
  - Irrigation techniques
  - Crop diseases

### 📊 Smart Decision Support
- Uses historical agricultural data
- Helps farmers improve yield and reduce crop failure risks
- Provides intelligent insights for better farming practices
